Class PermissionDeniedError

Raised when the caller does not have permission to run an operation.

Inherits From: OpError

Aliases:

  • Class tf.compat.v1.errors.PermissionDeniedError
  • Class tf.compat.v2.errors.PermissionDeniedError

For example, running the tf.WholeFileReader.read operation could raise PermissionDeniedError if it receives the name of a file for which the user does not have the read file permission.

error_code

The integer error code that describes the error.

The error message that describes the error.

node_def

The NodeDef proto representing the op that failed.

op

The operation that failed, if known.

N.B. If the failed op was synthesized at runtime, e.g. a Send or Recv op, there will be no corresponding tf.Operation object. In that case, this will return None, and you should instead use the tf.errors.OpError.node_def to discover information about the op.

Returns:

The Operation that failed, or None.
