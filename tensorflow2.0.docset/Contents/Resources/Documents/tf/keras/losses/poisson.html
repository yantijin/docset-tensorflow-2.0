tf.keras.losses.poisson

Computes the Poisson loss between y_true and y_pred.

python tf.keras.losses.poisson( y_true, y_pred )

The Poisson loss is the mean of the elements of the Tensor y_pred - y_true * log(y_pred).

Usage:

python loss = tf.keras.losses.poisson([1.4, 9.3, 2.2], [4.3, 8.2, 12.2]) print('Loss: ', loss.numpy()) # Loss: -0.8045559

Args:

  • y_true: Tensor of true targets.
  • y_pred: Tensor of predicted targets.

Returns:

A Tensor with the mean Poisson loss.

Raises:

  • InvalidArgumentError: If y_true and y_pred have incompatible shapes.
