tf.keras.backend.flatten

View source

Flatten a tensor.

Aliases:

  • tf.compat.v1.keras.backend.flatten
  • tf.compat.v2.keras.backend.flatten

python tf.keras.backend.flatten(x)

Arguments:

  • x: A tensor or variable.

Returns:

A tensor, reshaped into 1-D

Example:

```python

b = tf.constant([[1, 2], [3, 4]]) b <tf.Tensor: id=102, shape=(2, 2), dtype=int32, numpy= array([[1, 2], [3, 4]], dtype=int32)> tf.keras.backend.flatten(b) <tf.Tensor: id=105, shape=(4,), dtype=int32, numpy=array([1, 2, 3, 4], dtype=int32)> ```
