tf.keras.backend.int_shape

View source

Returns the shape of tensor or variable as a tuple of int or None entries.

Aliases:

  • tf.compat.v1.keras.backend.int_shape
  • tf.compat.v2.keras.backend.int_shape

python tf.keras.backend.int_shape(x)

Arguments:

  • x: Tensor or variable.

Returns:

A tuple of integers (or None entries).

Examples:

python >>> from keras import backend as K >>> input = K.placeholder(shape=(2, 4, 5)) >>> K.int_shape(input) (2, 4, 5) >>> val = np.array([[1, 2], [3, 4]]) >>> kvar = K.variable(value=val) >>> K.int_shape(kvar) (2, 2)
