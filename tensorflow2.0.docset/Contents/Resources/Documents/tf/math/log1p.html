tf.math.log1p

Aliases:

I.e., \(y = \log_e (1 + x)\).

Args:

  • x: A Tensor. Must be one of the following types: bfloat16, half, float32, float64, complex64, complex128.
  • name: A name for the operation (optional).

Returns:
