python __call__( shape, dtype=None, partition_info=None )

Returns a tensor object initialized as specified by the initializer.

Args:

  • shape: Shape of the tensor.
  • dtype: Optional dtype of the tensor. If not provided use the initializer dtype.
  • partition_info: Optional information about the possible partitioning of a tensor.

from_config

python @classmethod from_config( cls, config )

Instantiates an initializer from a configuration dictionary.

Example:

python initializer = RandomUniform(-1, 1) config = initializer.get_config() initializer = RandomUniform.from_config(config)

Args:

  • config: A Python dictionary. It will typically be the output of get_config.

Returns:

An Initializer instance.
