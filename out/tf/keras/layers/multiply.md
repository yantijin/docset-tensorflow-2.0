### Aliases:

* `tf.compat.v1.keras.layers.multiply`

``` python

    inputs,
    **kwargs

```



<!-- Placeholder for "Used in" -->


#### Arguments:


* <b>`inputs`</b>: A list of input tensors (at least 2).
* <b>`**kwargs`</b>: Standard layer keyword arguments.


#### Returns:

A tensor, the element-wise product of the inputs.
